How live refresh works:
When you click ↺ Refresh Live Data, the dashboard reads 3 sheets from your Google Sheet:
• Comparison sheet → poles, line work, DTR, financial %
• Overview sheet → supply amount, erection amount, overall %

When you update your Google Sheet:
1. Update the values in your sheet as usual
2. Come back to this dashboard
3. Click ↺ Refresh Live Data — all numbers update instantly ✅
How to Make Sure Refresh Works
1
Your sheet must be published to the web. Go to File → Share → Publish to web → Publish
2
This dashboard must be opened from a web server URL (like your GitHub Pages link), not from a local file on your computer
3
Your GitHub Pages URL should look like:
https://yourusername.github.io/rdss-dashboard
4
Once opened from GitHub Pages, click ↺ Refresh Live Data — it should show ✅ LIVE in the status bar
